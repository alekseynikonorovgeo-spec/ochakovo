📐 Калькулятор площади
Считайте площадь захватки сложной формы. Добавляйте фигуры (прямоугольник, трапеция, треугольник, круг) — можно прибавлять и вычитать (например, минус колодцы или островки).
💡 «📏 Полный заезд» — для симметричных заездов с одинаковыми радиусами. Промеры = полная ширина.
💡 «📐 Половина заезда» — для несимметричных заездов с разными радиусами. Делайте 2 половины (левая + правая), промеры = от оси до кромки.
💡 3 промера vs 4 промера — 3 хватит для короткого/плавного заезда; 4 точнее для длинных и сильно изогнутых (точность ≈ ±2-3% против ±5%).
Итого
Формулы:
• Прямоугольник: S = a × b
• Трапеция: S = (a + b) / 2 × h
• Треугольник: S = a × h / 2
• Круг: S = π × r²
• Сектор круга: S = π × r² × (угол / 360) — для четверти 90°, для полукруга 180°
• Скругление угла: S = r² × (1 − π/4) ≈ 0,2146 × r² — то что «съедает» дуга в углу
• Полоса по промерам: S = L × (w₁ + 2·w₂ + w₃) / 4 — приближение по 3 промерам (метод трапеций)
• Половина заезда: та же формула — для каждой стороны при разных радиусах

Знак «+» прибавляет, «−» вычитает из общей площади (для колодцев, островков, скруглений).

ℹ️ Плотность щебня

Плотность щебня по умолчанию принята 1,30 т/м³ — это усреднённый показатель.

Реальная плотность зависит от типа щебня (гранитный/известняковый/гравийный), фракции и партии — она указана в паспорте качества карьера.

Для точного учёта уточните плотность в ПТО и установите её:
• В профиле (⚙ Настройки расчёта) — применится ко всем приёмам
• В поле «Плотность» конкретного приёма — для разовых случаев

Если оба поля пусты — расчёт пойдёт по 1,30.
