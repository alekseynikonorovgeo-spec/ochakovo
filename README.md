# ОЧАКОВО — Конструктор отчёта (PWA)

Конструктор сменных отчётов прораба ПК ОЧАКОВО.  
Работает в браузере **и устанавливается на телефон как обычное приложение**.

---

## 📦 Что внутри

```
ochakovo-pwa/
├─ index.html             ← главный файл программы
├─ manifest.json          ← паспорт приложения (название, иконки)
├─ sw.js                  ← service worker (для офлайн-работы)
└─ icons/
   ├─ icon.svg            ← исходник иконки
   ├─ icon-192.png        ← иконка для меню (Android, iOS)
   ├─ icon-512.png        ← большая иконка
   └─ favicon-32.png      ← favicon для вкладки браузера
```

## 🚀 Как опубликовать (один раз)

### Вариант А — через GitHub Pages (рекомендуется)

1. Зарегистрироваться на **github.com** (бесплатно, 2 минуты)
2. Создать новый репозиторий:
   - Кнопка **«+»** справа сверху → **New repository**
   - Имя: `ochakovo` (или любое)
   - Поставить галочку **Public**
   - Кнопка **Create repository**
3. Загрузить файлы:
   - **Add file** → **Upload files**
   - Перетащить ВСЕ файлы из этой папки (включая папку `icons/`)
   - Внизу — **Commit changes**
4. Включить хостинг:
   - Вкладка **Settings** репозитория
   - Слева — **Pages**
   - Source: **Deploy from a branch**
   - Branch: `main`, папка `/ (root)` → **Save**
5. Через 1-2 минуты появится ссылка типа:  
   `https://ваш-ник.github.io/ochakovo/`

Эту ссылку даёте прорабам.

### Вариант Б — через Netlify Drop (без регистрации)

1. Открыть https://app.netlify.com/drop
2. Перетащить папку `ochakovo-pwa` в браузер
3. Получить ссылку типа `https://random-name.netlify.app`

## 📱 Как прорабу установить на телефон

### Android (Chrome / Яндекс.Браузер)
1. Открыть ссылку в браузере
2. Снизу появится баннер **«Установить ОЧАКОВО»** → нажать
3. Если баннера нет: меню (⋮) → **Установить приложение**
4. Иконка появится на главном экране — как обычная программа

### iPhone (Safari)
1. Открыть ссылку **обязательно в Safari** (Chrome на iPhone не подходит!)
2. Внизу — кнопка **«Поделиться»** (квадрат со стрелкой)
3. Промотать вниз → **«На экран Домой»**
4. Нажать **«Добавить»**
5. Иконка появится на главном экране

После установки приложение запускается **без браузера**, в полноэкранном режиме.

## 🔄 Как обновить программу (когда выйдет новая версия)

1. Зайти в репозиторий на GitHub
2. Открыть `index.html` → кнопка-карандаш (Edit)
3. Заменить содержимое на новое → **Commit changes**
4. **ВАЖНО**: открыть `sw.js` → найти `ochakovo-v1` → поменять на `ochakovo-v2` (или v3, v4...)
5. **Commit changes**

Прорабы получат обновление автоматически при следующем открытии с интернетом.

## 🌐 Работа БЕЗ интернета

- При **первом** открытии — интернет нужен (скачать файлы программы)
- Дальше работает **полностью офлайн**:
  - Создание отчётов ✓
  - Калькулятор асфальта ✓
  - Калькулятор площади ✓
  - Сохранение черновиков ✓
  - Импорт/экспорт шаблонов JSON ✓
- При **обновлении** программы — интернет нужен один раз (для скачивания)

---

## 💾 Данные прораба

Все данные хранятся **на телефоне прораба** (localStorage браузера):
- Черновики смен
- Пользовательские шаблоны
- Профиль (ФИО, телефон, участок)
- Кастомные объекты

На сервере **ничего не хранится** — только статические HTML/CSS/JS.

---

## 🆘 Если что-то не работает

- **Не появляется баннер установки** (Android) — обновить браузер до последней версии
- **Не работает офлайн** — нужно открыть приложение хотя бы раз с интернетом
- **Старая версия не обновляется** — проверить что в `sw.js` поменяли номер версии
- **iPhone не предлагает установить** — обязательно открывать в **Safari**, не в Chrome
